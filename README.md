# ImageViewerPlusPlus
Advanced image viewer I made awhile back that supports 27 image formats across 30 file extensions 

## Supported File Extensions/Types
 - `.png`
 - `.jpg`
 - `.jpeg`
 - `.gif`
 - `.bmp`
 - `.tiff`
 - `.tif`
 - `.webp`
 - `.ico`
 - `.heic`
 - `.heif`
 - `.avif`
 - `.jp2`
 - `.jpx`
 - `.psd`
 - `.pcx`
 - `.tga`
 - `.exif`
 - `.wmf`
 - `.emf`
 - `.dib`
 - `.pbm`
 - `.pgm`
 - `.ppm`
 - `.pnm`
 - `.sgi`
 - `.im`
 - `.cur`
 - `.xbm`
 - `.xpm`
